# R + RStudio

Rという統計処理パッケージは、RStudioという統合環境を通して利用するのが便利です。Rの使い方については参考URLも参照してください。

## 所要時間

- 10分

## 手順

### Rのインストール

- [Rのホームページ](https://www.r-project.org/)にいく
  - Download R > リストから`Japan`を探す > URLを選択
  - Download R for Windows > base > Download N.N.N R for Windows を選択 > ダウンロード > 実行
  - 日本語 > （すべて）次へ > 完了
  

### RStudioのインストール

- [R Studioのダウンロードページ](https://rstudio.com/products/rstudio/download/)にいく
  - RStudio Desktop Open Source License > Download > Download RStudio for Windows を選択 > ダウンロード > 実行
  - （すべて）次へ > 完了
