いつクラスターを使うのか When should I use the cluster?

  • 研究室PCの容量を超えたデータ処理 When you need to process a data which is beyond a lab PC's storage
  • 数時間かかるデータ処理 When you have a process that can continue for several hours
  • GPUが必要なデータ処理 When you need a GPU to process a data

クラスターを使用する前に Before you use the cluster

  • 研究室PCは意外と高性能です。 Lab PC's spec is not bad actually
  • 研究室PCでサンプルデータの処理を完成させましょう Make sure to complete your program with a small sample
  • 小さなデータから大きなデータに移行するのは良い習慣です。It's a good practice to move from a small dataset to a large dataset
  • 必ず研究室クラスターの使い方を読んでください。Make sure to read the How to ues lab cluster very carefully
